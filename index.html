About this page — what it is, how it works, and the fine print.

What this is: a rolling, searchable archive of newly published papers across 58 journals — regional science, urban/spatial economics, rural & land studies, and agricultural & environmental economics — with abstracts inline where openly available and a link to the version of record for every paper. It is a personal literature-tracking tool, shared publicly in case it is useful to others.

How it updates: every Monday morning, an automated Claude (Anthropic) session sweeps the journals' public pages and open bibliographic sources (SpringerLink, SAGE, Oxford University Press, RePEc/IDEAS, Crossref mirrors), adds new papers with a NEW badge, tags them by theme, refreshes the deep-read shortlist, and republishes this page via GitHub Pages. Each update is a Git commit, so the full history is preserved in the repository.

Self-maintenance: the site fetches missing abstracts automatically (via the owner's devices or the weekly update) and republishes every Monday. Everything here runs without manual upkeep.

How to use
  • Search box — matches words anywhere in titles, authors, abstracts, keywords, journal names, and theme labels; multiple words must all match.
  • Theme chips and Journal chips — tap the row label (THEME ▸ / JOURNAL ▸) to unfold all options; one tap filters, counts shown. The small green tags on each paper filter by theme too.
  • Published: presets (last month / 3 / 12 months) or exact from–to dates; Update: shows only papers that arrived in one specific Monday update; NEW badges mark the latest batch.
  • Sort — by journal, newest/oldest publication date, or title; 'research only' hides book reviews and corrigenda; '› more' expands long abstracts.
  • The black DOI button opens the version of record (use institutional library access for paywalled full text); Spreadsheet (top bar) downloads the archive as .xlsx and Data as raw JSON.
